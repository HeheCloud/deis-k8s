# Deis Kubernetes (k8s)

Kubernetes for use in the [Deis@HeheCloud](http://hehecloud.com/) open source PaaS.

Please add any [issues](https://github.com/HeheCloud/deis-k8s/issues) you find with this software to the Project.

## Usage

```
$ deisctl install k8s
$ deisctl start k8s
```
